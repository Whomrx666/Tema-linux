# Tema linux
![Tema-linux](Tema-linux.png)

Linux Theme for Termux

## Instalations
```
$ pkg update && pkg upgrade
$ pkg install python
$ pkg install git
$ git clone https://github.com/Whomrx666/Tema-linux
$ cd Tema-linux
$ python Tema-linux.py

```
